CSS

Cascading Style Sheets (CSS) is a style sheet language used for describing the presentation of a document written in a markup language such as HTML or XML (including XML dialects such as SVG, MathML or XHTML).[1] CSS is a cornerstone technology of the World Wide Web, alongside HTML and JavaScript.[2]

  • CSS의 등장(생활코딩 youtube 동영상 설명)

* css를 사용하는 이유 : 글씨색 표현 방법 : <style> 태그로 삽입

과거에는 font 태그로 글씨색을 표현 하였으나 현재는 <style> 태그를 <head> 사이에 직접 삽입해서 CSS 선택자를 적용하는 것이다. 디자인(글씨색 CSS)과 정보(HTML 태그)가 분리되어 편집자와 컴퓨터 모두 편리하게 되었다. 한 두개 정도야 직접 타이핑 하면 되지만 동일 코드가 1억개 있다고 치면 색 변경을 위하여 1억개를 모두 타이핑하는 것은 거의 불가능에 가깝다. css를 활용하면 color 단어 하나면 수정하면 가능하다.

아래 예시 처럼 <font> 태그에 color 속성을 사용해도 작동은 합니다. 하지만 작동이 되는 이유는 과거의 수 많은 웹문서를 보존하기 위함이지 웹표준은 아닙니다. 맨 아래 처럼 태그에 style 속성과 속성값을 따로 삽입하는 것이 웹표준입니다.

<h2>수메르 어원</h2>
<font color="red"><p>아카드인의 방언 : 아카드인이 메소포타미아 남부 지방 사람들을 아카드어로 부르던 말</p></font>
<p style="color:blue;">웅 상 기가 : 수메르인들이 수메르어로 자신들을 부르던 말, 검은 머리 사람들이라는 의미</p>

수메르어 어원

  • 시날(Shinar) : 성경에서 수메르를 지칭하는 방언
  • 신그르(Sngr) : 이집트 문명에서 수메르를 지칭하는 방언
  • 산하르(Sanhar) : 히타이트에서 수메르를 지칭하는 방언
<!DOCTYPE html>
<html>
  <head>
    <style>
      ul {
        color: red;
      }
    </style>
  </head>
  <body>
    <h2>수메르어 어원</h2>
      <ul>
        <li>시날(Shinar) : 성경에서 수메르를 지칭하는 방언</li>
        <li>신그르(Sngr) : 이집트 문명에서 수메르를 지칭하는 방언</li>
        <li>산하르(Sanhar) : 히타이트에서 수메르를 지칭하는 방언</li>
      </ul>
  </body>
</html>
