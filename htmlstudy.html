HTML

HyperText Markup Language (HTML) is a type of markup language.[1] It is used to make webpages. Webpages can include writing, links, pictures, and even sound and video. HTML tells web browsers what webpages should look like. HTML also adds meta information to webpages. Meta information is information about about a webpage e.g., the name of the person who created the page. Web browsers usually do not show meta information.

<html> 태그에 관하여...

  • <a>태그 : list의 약자로 즉, 목록을 만들 때 사용하며, ul 태그와 함께 사용된다.

    • 마우스를 올려보세요

    <a href="#" title="마우스를 올리면 | 부연 설명이 나와요">마우스를 올려보세요</a>

    이같이 링크에 마우스를 올리면 부연 설명이 나와요.

  • <li>태그 : list의 약자로 즉, 목록을 만들 때 사용하며, ul 태그와 함께 사용된다.
  • 태그이름 : 부등호 < 는 html 파일에 직접 사용불가. 따락서 < 기호를, &lt; 이렇게 표기하면 태그와 혼동되지 않고 부등호가 잘 표현됩니다.
  • <h1>태그 : 제목 태그로 한 페이지에 한 번만 사용한다.
  • <h1>태그 색상 : <h3 style="background-color:orange;">미래인류의 시간 여행 </h3>
  • &amp : 앰퍼샌드(&) 기호는, 위의 경우들에서 보는 것과 같이 이미 특수한 용도로 사용되고 있습니다. 그래서 & 기호 자체를 표현해 주기 위해서는 & 를, &amp; 이렇게 표기해 주어야 합니다.
  • > 태그 : &gt;
  • canvas 태그

    • canvas 태그는 픽셀 단위로 컨프롤 할 수 있어서 html 이 할 수 없는 작업을 가능하게 한다. 예를 들어 비디오도 조각으로 나누어 조작 가능함. 비트맵 이다. svg 는 벡터.

  • HTML5 Canvas 캔버스 라이브 강좌 #1_1분코딩

1. &nbsp;

[H     O  N  G]

- HTML에서 위와같이 글씨 사이에 공백을 여러번 입력해도 1개로 인식한다.

[H O N G]

- 그래서 공백을 여러개 넣고 싶을때는 &nbsp;를 여러번 입력한다.

[H&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;O&nbsp;&nbsp;N&nbsp;&nbsp;G]

 

2. &lt;

[< HONG >]

- HTML에서 위와같이 입력하면 태그[명령어]로 인식한다.

[ &lt; HONG &gt;]

- 그래서 <를 넣고 싶을때는 위와같이 &lt;를 입력한다.

 

3. &gt;

[< HONG >]

- HTML에서 위와같이 입력하면 태그[명령어]로 인식한다.

[ &lt; HONG &gt;]

- 그래서 >를 넣고 싶을때는 위와같이 &gt;를 입력한다.

 

4. <CENTER></CENTER>

[                          HONG                          ]

- HTML에서 위와같이 가운데 정렬을 하고 싶을 때 <CENTER>를 사용한다.

[<CENTER>HONG</CENTER>]

 

5. <HR/>

[ --------------------------------------------------]

- HTML에서 구분선을 긋고 싶을때 사용한다.

[<HR/>]

 

6. <I></I>

[HONG]

- HTML에서 이태리체 글씨로 표현하고 싶을때 사용한다.

[<I>HONG</I>]

 

7. <B></B>

[HONG]

- HTML에서 글씨를 두껍게 표현하고 싶을때 사용한다.

[<B>HONG</B>]

 

8. <U></U>

[HONG]

- HTML에서 글씨에 밑줄을 긋고 싶을때 사용한다.

[<U>HONG</U>]

 

9. <BR/>

[HO]

[NG]

- HTML에서 글씨를 아래로 내리고 싶을때 사용한다. [ ENTER키 효과 ]

[HO<BR/>NG]

 

10. <H1></H1>     <H2~H6>

[HONG_H1]

[HONG_H2]

[HONG_H3]

[HONG_H4]

[HONG_H5]

[HONG_H6]

[<H2>HONG_H2</H2>]

[<H3>HONG_H3</H3>]

[<H4>HONG_H4</H4>]

html 관련 용어

  • MDN 웹 문서(영어: MDN Web Docs, 이전 이름 모질라 개발자 네트워크(영어: Mozilla Developer Network), 모질라 개발자 센터(영어: Mozilla Developer Center))는 웹 표준과 모질라 프로젝트에 대한 개발 문서들이 담긴 모질라의 공식 웹사이트이다
  • DOM 이란? 문서 객체 모델(DOM, Document Object Model)은 XML이나 HTML 문서에 접근하기 위한 일종의 인터페이스입니다. 이 객체 모델은 문서 내의 모든 요소를 정의하고, 각각의 요소에 접근하는 방법을 제공합니다.
