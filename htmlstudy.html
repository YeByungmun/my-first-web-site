HTML 이란?

HyperText Markup Language (HTML) is a type of markup language.[1] It is used to make webpages. Webpages can include writing, links, pictures, and even sound and video. HTML tells web browsers what webpages should look like. HTML also adds meta information to webpages. Meta information is information about about a webpage e.g., the name of the person who created the page. Web browsers usually do not show meta information.

HTML is often used with Cascading Style Sheets (CSS) and JavaScript. CSS is used to change the way HTML looks. JavaScript tells websites how to behave. It can also change HTML and CSS.

HTML is made by the World Wide Web Consortium (W3C). There are many versions of HTML. As of September 2018, the current standard of HTML is called HTML 5 and is specifically at version 5.2.

1. &nbsp;

[H     O  N  G]

- HTML에서 위와같이 글씨 사이에 공백을 여러번 입력해도 1개로 인식한다.

[H O N G]

- 그래서 공백을 여러개 넣고 싶을때는 &nbsp;를 여러번 입력한다.

[H&nbsp;&nbsp;&nbsp;&nbsp;&nbsp;O&nbsp;&nbsp;N&nbsp;&nbsp;G]

 

2. &lt;

[< HONG >]

- HTML에서 위와같이 입력하면 태그[명령어]로 인식한다.

[ &lt; HONG &gt;]

- 그래서 <를 넣고 싶을때는 위와같이 &lt;를 입력한다.

 

3. &gt;

[< HONG >]

- HTML에서 위와같이 입력하면 태그[명령어]로 인식한다.

[ &lt; HONG &gt;]

- 그래서 >를 넣고 싶을때는 위와같이 &gt;를 입력한다.

 

4. <CENTER></CENTER>

[                          HONG                          ]

- HTML에서 위와같이 가운데 정렬을 하고 싶을 때 <CENTER>를 사용한다.

[<CENTER>HONG</CENTER>]

 

5. <HR/>

[ --------------------------------------------------]

- HTML에서 구분선을 긋고 싶을때 사용한다.

[<HR/>]

 

6. <I></I>

[HONG]

- HTML에서 이태리체 글씨로 표현하고 싶을때 사용한다.

[<I>HONG</I>]

 

7. <B></B>

[HONG]

- HTML에서 글씨를 두껍게 표현하고 싶을때 사용한다.

[<B>HONG</B>]

 

8. <U></U>

[HONG]

- HTML에서 글씨에 밑줄을 긋고 싶을때 사용한다.

[<U>HONG</U>]

 

9. <BR/>

[NG]
